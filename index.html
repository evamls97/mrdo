Una mascota virtual y un respiro para tu mente

Con esta aplicación de autocuidado emocional contarás con un acompañante muy especial para ayudarte a gestionar el estrés y la ansiedad. Todo mediante pequeñas actividades diarias o el diario personal, que están diseñadas para ayudarte a comprender mejor tus emociones y pensamientos.

```

¿Qué te espera dentro?

🟠 Mascota

Adopta a una pequeña bolita de enredos y cuídala. Podrás acariciarlo y darle de comer mientras te acompaña con tus actividades diarias.

🏆 Actividades

Completa sencillas actividades diarias para hacer feliz a tu mascota. ¡Aprenderás un montón sobre hábitos saludables!

✏️ Diario

Este será tu espacio personal. Podrás escribir lo que quieras cada día. No importa si son notas para recordar algo o si es solo para desahogarte... Aquí decides únicamente tú.
